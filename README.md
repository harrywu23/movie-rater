# Movie Rater

A simple web application to rate movies, built with React.

## Features

- Add movies to your list.  
- Rate movies with a score.  
- View your movie ratings.  
- Responsive design for desktop and mobile.  

### Prerequisites

- Node.js (v16+)  
- npm  

### Installation

1. Clone the repository:  
   ```bash
   git clone https://github.com/harrywu23/movie-rater.git
   cd movie-rater

   npm install
   npm run dev
   Open http://localhost:3000 to view the app in your browser.
